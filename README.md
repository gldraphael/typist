# typist

## Quickstart

### One time setup

1. Install [.NET Core SDK 3.1.101 (LTS)](https://dotnet.microsoft.com/download/dotnet-core/3.1) for your platform.
1. Clone this repo:

    ```sh
    git clone git@github.com:gldraphael/typist.git
    ```

### Running the app

```sh
cd src/TypistApp
dotnet run
```
